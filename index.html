Lena - The Language

Lena is a web based language which compiles down to web resources such as HTML, CSS e.t.c. It has an easy to understand syntax so the process of building UIs can be very easily automated. It used components to bring in the flexibility of reusability, prop drilling.

Powerful Tools

With a rich set of features and tools, Lena enables developers to build complex applications with ease. It can build beutiful UIs using the power of tailwind classes. Fun Fact: This site is built in lena too!

Written in Rust!

Lena's compiler is written in Rust which is a memory safe and very fase language, so we can bring in all the type safe features of rust in the compiler in the upcoming time.

Easy to Learn

Lena's clean and intuitive syntax makes it easy to write and understand code, speeding up the development process.

Learn Lena Syntax

Basic Component

Creating a basic component in Lena is straightforward. Here's how you can define a simple button component:


    Button {
        props: {
            class: "bg-blue-500 text-white p-2 rounded-lg"
        },
        "Click Me"
    }

Using Props

Props allow you to pass data into your components. Here's an example of a component that takes a 'title' prop:


    Card {
        props: {
            title: "Card Title",
            class: "border p-4 rounded-lg"
        },
        h3 {
            props: {
                class: "text-xl font-bold"
            },
            "Hello"
        }
        p {
            "This is a card component."
        }
    }

Nested Components

Lena allows you to nest components within each other. Here's an example:


    Container {
        props: {
            class: "p-4"
        },
        Header {
            h1 {
                "Welcome"
            }
        }
        Content {
            p {
                "This is the content area."
            }
        }
        Footer {
            p {
                "Footer information here."
            }
        }
    }
